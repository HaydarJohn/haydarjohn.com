# The start of a journet: Blog Website

I do not watch any fancy videos or stuff. I am using very old school technique called reading. I am using [this](https://www.w3schools.com/) website for guide. Lets start by basics.

So I wondered why is there two different linking mechanisims. If you think about it both src and href achieve the same thing. point to something. Why cant we use them interchangeably?
Creator of html Tim Berners-Lee made browser and did not support images. Some other guy added \<img> tag and used src for pointing. there was no governing body for web at the time.
So yeah then rest is wild west until standartization. But if i understand it correctly if you use a referece either your files or other websites you use href (like table of contents. just tells what is where) and if you want to load image or scripts you use src(These can fail. So loading image or script is not just pointing so use src.)
